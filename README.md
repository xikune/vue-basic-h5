>  模板基于 vue-cli4 和 Vant-ui 搭建，进行大型 H5 项目开发最佳实践方案

## 安装

```sh
# 克隆项目
git clone git@github.com:FreeDemons/vue-basic-h5.git

# 进入项目目录
cd vue-basic-h5

# 安装依赖
npm install

# 建议不要用 cnpm 安装 会有各种诡异的bug 可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org

# 本地开发 启动项目
npm run serve
```
